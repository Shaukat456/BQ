## ⏱️ Algorithmic Complexity

_"EFFICIENCY IS CRUCIAL, Boss! Every mad scientist must understand the cost of their experiments! xD"_

### Time Complexity

Time complexity measures how runtime grows as input size increases. We use Big O notation to express this relationship.

| Notation   | Name        | Description                              | Example                         |
| ---------- | ----------- | ---------------------------------------- | ------------------------------- |
| O(1)       | Constant    | Runtime is independent of input size     | Array access, hash table lookup |
| O(log n)   | Logarithmic | Runtime grows logarithmically with input | Binary search                   |
| O(n)       | Linear      | Runtime grows linearly with input        | Linear search                   |
| O(n log n) | Log-Linear  | Runtime grows by n log n                 | Efficient sorting (merge, heap) |
| O(n²)      | Quadratic   | Runtime grows with square of input       | Bubble sort, insertion sort     |
| O(2ⁿ)      | Exponential | Runtime doubles with each added element  | Recursive Fibonacci             |

### Space Complexity

Space complexity measures how memory usage grows as input size increases.

| Space Complexity | Example                         |
| ---------------- | ------------------------------- |
| O(1)             | Iterative factorial calculation |
| O(n)             | Storing an array of size n      |
| O(n²)            | Storing an n×n matrix           |

### Common Python Operations Complexity

```python
# List operations
my_list = [1, 2, 3, 4, 5]
element = my_list[0]       # O(1) - Accessing by index
my_list.append(6)          # O(1) - Adding to end
my_list.pop()              # O(1) - Removing from end
my_list.insert(0, 0)       # O(n) - Insertion requires shifting elements
my_list.pop(0)             # O(n) - Removal from beginning requires shifting

# Dictionary operations
my_dict = {'a': 1, 'b': 2}
value = my_dict['a']       # O(1) average - Hash table lookup
my_dict['c'] = 3           # O(1) average - Adding a new key-value pair
'c' in my_dict             # O(1) average - Checking existence

# Set operations
my_set = {1, 2, 3}
4 in my_set                # O(1) average - Membership check
my_set.add(4)              # O(1) average - Adding an element
my_set.remove(1)           # O(1) average - Removing an element
```

## 🧪 Advanced Experimentation Techniques

### Testing Your Creations

```python
# Example of a simple test function
def test_function(function_to_test, test_cases):
    """Run test cases against a function and report results"""
    results = []
    for inputs, expected in test_cases:
        actual = function_to_test(*inputs)
        success = actual == expected
        results.append({
            'inputs': inputs,
            'expected': expected,
            'actual': actual,
            'success': success
        })

    # Calculate success rate
    success_count = sum(r['success'] for r in results)
    rate = success_count / len(results) * 100

    print(f"Success rate: {rate:.2f}% ({success_count}/{len(results)})")

    # Print failed tests
    failed = [r for r in results if not r['success']]
    if failed:
        print("\nFailed tests:")
        for i, test in enumerate(failed):
            print(f"{i+1}. Input: {test['inputs']}")
            print(f"   Expected: {test['expected']}")
            print(f"   Actual: {test['actual']}\n")
```

### Performance Optimization and Analysis

```python
# Example timing decorator with complexity annotation
import time
import math

def measure_performance(expected_complexity="O(n)"):
    """Decorator to measure and analyze function performance"""
    def decorator(func):
        def wrapper(*args, **kwargs):
            # Identify the first numeric argument as input size
            input_size = next((len(arg) for arg in args if hasattr(arg, '__len__')), 0)

            start_time = time.time()
            result = func(*args, **kwargs)
            elapsed_time = time.time() - start_time

            # Simple complexity estimation based on running time
            print(f"{func.__name__} with input size {input_size}:")
            print(f"  - Time: {elapsed_time:.6f} seconds")
            print(f"  - Expected complexity: {expected_complexity}")

            if input_size > 0:
                # Calculate expected time ratios for common complexity classes
                if expected_complexity == "O(n)":
                    expected_ratio = input_size
                elif expected_complexity == "O(n²)":
                    expected_ratio = input_size ** 2
                elif expected_complexity == "O(log n)":
                    expected_ratio = math.log2(input_size) if input_size > 0 else 0
                elif expected_complexity == "O(n log n)":
                    expected_ratio = input_size * math.log2(input_size) if input_size > 0 else 0
                else:
                    expected_ratio = 0

                print(f"  - Time/Size Ratio: {elapsed_time/input_size:.8f} seconds per element")

            return result
        return wrapper
    return decorator

# Usage
@measure_performance("O(n log n)")
def merge_sort(arr):
    # Implementation here
    pass
```

### Memory Usage Analysis

```python
# Example memory usage tracking
import tracemalloc

def measure_memory_usage(func):
    """Decorator to measure memory usage of a function"""
    def wrapper(*args, **kwargs):
        tracemalloc.start()
        result = func(*args, **kwargs)
        current, peak = tracemalloc.get_traced_memory()
        tracemalloc.stop()

        print(f"{func.__name__} memory usage:")
        print(f"  - Current: {current / 1024:.2f} KB")
        print(f"  - Peak: {peak / 1024:.2f} KB")

        return result
    return wrapper

# Usage
@measure_memory_usage
def create_large_data_structure():
    # Code that creates a data structure
    pass
```
